# React E‑Commerce

A responsive, single‑page e‑commerce front‑end built with **React**, **Vite**, and **Tailwind CSS**. The application consumes the [Free EscuelaJS API](https://api.escuelajs.co/api/v1/) to display categories, products and product details. It includes search functionality, category browsing, a product details page with image gallery, and a contact form.

## 🚀 Features

- Browse product categories and view up to 10 on the home page
- Search input (UI only)
- View a list of products with basic information
- Product detail page with image slider, price and description
- Order form component stub (`TakeOrder`) for future integration
- Mobile‑friendly responsive layout built with Tailwind CSS
- Navigation with **React Router v7** including 404 page
- Global loading indicator using Axios interceptors
- Contact page with static form and company details

## 🧰 Technologies Used

- [React 18](https://reactjs.org/)
- [Vite](https://vitejs.dev/) (development server & bundler)
- [Tailwind CSS](https://tailwindcss.com/) for utility‑first styling
- [Axios](https://axios-http.com/) for HTTP requests
- [React Router](https://reactrouter.com/) for client‑side routing
- [Swiper](https://swiperjs.com/) (carousel) via `swiper` dependency
- [React Toastify](https://fkhadra.github.io/react-toastify/) for notifications (used in future enhancements)

## ⚙️ Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) 18+ or equivalent
- npm or yarn

### Installation

```bash
# clone the repo
git clone https://github.com/your-username/react-ecommerce.git
cd react-ecommerce

# install dependencies using npm or yarn
npm install
# or
# yarn
```

### Development

Start the Vite development server:

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser. The app supports hot module replacement and will reload when you modify files.

### Build for Production

```bash
npm run build
```

The production assets are generated in the `dist/` directory. You can preview them with:

```bash
npm run preview
```

## 📦 API

This project uses the public [EscuelaJS API](https://api.escuelajs.co/api/v1/). All HTTP requests are handled through the `apiRequest` instance in `src/Axios.js` which is preconfigured with:

```js
baseURL: "https://api.escuelajs.co/api/v1/",
timeout: 30000,
headers: { "Content-Type": "application/json" }
```

Endpoints consumed include:

- `/categories` – list of categories
- `/products` – product listing
- `/products/:id` – product details

Feel free to replace the base URL with your own backend when ready.
